Overview (Tong quan)

Tax Vietnam allows users to use tax codes finding company information. With this app, users can fill the tax code in the Contact form, the system automatically fills in the name, phone number.

Khi tao moi khach hang hoac nha cung cap, neu la cong ty thi chi can dien ma so thue, click ra ngoai -> Tu dien cac thong tin nhu ten, dia chi, tinh thanh...
